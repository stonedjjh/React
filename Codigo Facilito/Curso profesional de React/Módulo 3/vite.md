# Vite

Vite es una herramienta de construcción (build tool) que proporciona una experiencia de desarrollo frontend más rápida y optimizada. Utiliza módulos ES nativos en el navegador para acelerar el proceso de desarrollo y ofrece una configuración mínima para comenzar a trabajar rápidamente.

## Características principales

- **Desarrollo rápido**: Vite utiliza módulos ES nativos, lo que permite una recarga instantánea del navegador y tiempos de arranque rápidos.

- **Construcción optimizada**: Vite utiliza Rollup para construir y optimizar el código para producción, lo que resulta en paquetes más pequeños y eficientes.

- **Soporte para múltiples frameworks**: Vite es compatible con varios frameworks populares como React, Vue y Svelte, lo que facilita su integración en proyectos existentes.

- **Plugins**: Vite tiene un sistema de plugins que permite extender su funcionalidad y personalizar el proceso de construcción.

## Cómo funciona Vite

- Instant Server Start:
  Vite inicia un servidor de desarrollo en cuestión de milisegundos utilizando módulos ES nativos.

- Hot Module Replacement (HMR):
  Vite ofrece una experiencia de desarrollo fluida con HMR, lo que permite actualizar los módulos en tiempo real sin recargar la página completa.

- Rich Features
  Vite incluye características como soporte para TypeScript, CSS preprocesadores y más, facilitando el desarrollo moderno.

- Optimized Build:
  Vite utiliza Rollup para empaquetar y optimizar el código para producción, asegurando un rendimiento óptimo.

## Creando Proyectos con Vite

Para crear un nuevo proyecto con Vite, puedes utilizar el siguiente comando en la terminal:

```bash
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
```
