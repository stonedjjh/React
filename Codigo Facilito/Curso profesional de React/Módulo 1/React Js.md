# React Js

## Características

- Una librería para el desarrollo de frontend moderno.

- Se basa en componentes.

- Es declarativa.

- Es rápida, eficiente y fácil de aprender.

## React Js no es

- No es un framework.

- No es una herramienta de backend.

- No es un reemplazo de JavaScript

- No es una solución completa.
