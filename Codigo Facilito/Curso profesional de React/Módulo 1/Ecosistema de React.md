# Ecosistema de React

## Librerías Principales

- **Vite**: es un bundler que proporciona una experiencia de desarrollo frontend más rápida a través de módulos nativos ES.

- **Redux Toolkit**: es el conjunto de herramientas oficial y recomendado por Redux para simplificar las tareas de desarrollo y la lógica estándar de Redux, incluyendo la gestión de estados complejos.

- **Zustand**: es una librería de gestión de estados simple, rápida y escalable que utiliza hooks de React para gestionar el estado de manera eficiente y concisa.

- **React Router**: es una librería para manejar la navegación en aplicaciones React de una sola página (SPA).

- **SWR**: es una estrategia de recuperación de datos (data fetching) basada en hooks de React que se utiliza para obtener, almacenar en caché y revalidar datos de manera eficiente.

- **Axios**: es una librería para hacer solicitudes HTTP desde el navegador o Node.js basada en promesas.

- **React Hook Form**: es una librería para la gestión y validación de formularios que se centra en un rendimiento sin re-renderizados innecesarios.

- **Tailwindcss**: es un framework CSS de utilidad que permite construir diseños personalizados rápidamente directamente en el marcado HTML.

- **Styled Components**: es una librería que permite escribir código CSS dentro de JavaScript (CSS-in-JS) usando plantillas de etiquetas para estilizar componentes de React.

- **React Spring**: es una librería de animación basada en la física (physics-based) que es flexible y permite animaciones fluidas y declarativas.

- **React Testing Library**: es un conjunto de utilidades que facilita las pruebas de componentes de React centrándose en probar el comportamiento del usuario final.
